## sorted()
- 새로운 정렬된 목록 반환하며, 원래 목록은 영향 x
~~~
  sorted([5, 2, 3, 1, 4])
  
  [1, 2, 3, 4, 5]
  
  a = [5, 2, 3, 1, 4]
 ~~~~



## sort()
- 리스트를 정렬하고 목록 인덱스 변경하고 none 반환
~~~
  a.sort()
  a
  [1, 2, 3, 4, 5]
~~~

## 차이
- sorted() : 문자열, 튜플, 딕셔너리, 제너레이터 등 모든 요소가 포함된 반복 가능한 객체를 정렬하여 반환
- list를 변경하려면 list.sort()를 사용하고, 새로운 정렬된 객체를 원하면 sorted()를 사용
